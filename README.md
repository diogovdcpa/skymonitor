# skymonitor
SkyMonitor é um script de automação desenvolvido para interagir com as APIs do Skyhigh (CASB/DLP). Ele monitora proativamente logs de incidentes, filtra anomalias com base em regras personalizadas e agiliza o tempo de resposta (MTTR) através de alertas estruturados e logs detalhados de eventos de segurança.
